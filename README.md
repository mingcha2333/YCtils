# YCtils

**YCtils** 是一款基于 **Minecraft 1.21.3**、**Fabric** 开发的辅助模组，专为 **遗城** 服务器设计。  

## 功能概览（在做了在做了，已完成的会打钩标记）

### 主城
- 钓鱼倒计时 & 渔获统计
- 挂机收益统计
- 资源总数 HUD
- 种植收获倒计时
- 当前排队情况 & 地牢开启弹窗提示（到达队列尽头时）

### 地牢
- 更便捷的小地图
- 当前房间剩余未开启箱子总数
- 地牢难度 / 当前状态 / 剩余时间 HUD
- 队友状态显示
- 队伍 / 个人 DPS 统计
- 地牢内资源统计
- 装备稀有度背景
- 个人属性显示 HUD
- 地牢进度 & 用时显示
- 周围有钥匙 / 出口时 Title 提示
- 谜题解谜辅助（平衡性原因，暂未加入日程）
- 更详细的结算信息

### BOSS
- BOSS 血条旁显示剩余血量
- BOSS 状态提示

### 社交
- 添加关注的人
- 关注的人上线 / 离开提示

### 娱乐
- 播放音乐
- 内置贪吃蛇小游戏

## 安装要求

- Minecraft 1.21.3
- Fabric Loader 0.18.6
- Fabric API 0.114.1+1.21.3

## 数据收集声明

YCtils 仅会在你登录遗城服务器时，向模组作者的服务器发送以下信息：

- 你的游戏 ID
- 当前使用的模组版本

这些数据仅用于：
- 统计模组使用人数 ~（让作者知道有人在用他做的小破模组，从而激励作者）~
- 检查版本更新情况
模组不会收集任何其他信息。
